.TH gnome-appearance-properties 1 "August 24, 2009" "version 0.1"
.SH NAME
gnome\-appearance\-properties \- a graphical application for customizing the look and feel of the Gnome Desktop.
.SH SYNOPSIS
.B gnome\-appearance\-properties [options] [wallpaper]
.SH DESCRIPTION
gnome\-appearance\-properties is a graphical preferences tool, that lets you to configure various aspects of how your desktop looks, including Themes, Backgrounds, Fonts, Icons, Interfaces and Effects.
.SH OPTIONS
Help Options:
.TP
.B \-h or \-\-help
Show these help options and exit
.PP
Application Options:
.TP
.B \-i or \-\-install\-theme=filename
Specify the filename of a theme to install
.TP
.B \-p or \-\-show\-page=page             
Specify the name of the page to show (theme|background|fonts|interface)
.TP
.B \-\-display=DISPLAY                
X display to use
